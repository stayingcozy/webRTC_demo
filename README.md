# webRTC_demo

based off Fireship's "WebRTC in 100 Seconds // Build a Video Chat app from Scratch"

built with vite as vanilla js project. See raspberrypi branch for raspberry pi specific code.

## Setup
```
git clone <this-repo>
npm install
```

## Run
```
npm run dev
```
